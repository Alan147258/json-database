# json-database

### 项目介绍
基于nodejs，通过操作json文件实现的迷你数据库  

项目地址: [http://alan147258.gitee.io/json-database](http://alan147258.gitee.io/json-database)    


### 使用举例

```
git clone http://alan147258.gitee.io/json-database
```

```
cd json-database
```

```
node demo.js
```
