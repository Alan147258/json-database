# json-database

### 项目介绍
基于nodejs，通过操作json文件实现的迷你数据库  

项目地址: [https://github.com/Alan147258/json-database](https://github.com/Alan147258/json-database)    


### 使用举例 
可直接查看或运行demo.js
```
git clone https://github.com/Alan147258/json-database
```

```
cd json-database
```

```
node demo.js
```
